# Coeffects Bibliography

Inspired by the collaborative [effects bibliography](https://github.com/yallop/effects-bibliography),
this is a collection of research papers and other resources related to coeffect tracking
in programming languages theory and implementation.

### Contributing guidelines

* If you don't have push access, you can [create a pull request](https://github.com/plclub/coeffects-bibliography/pulls).
* Add an entry to this file **and** the `coeffects.bib` file,
  both sorted by date in newest → oldest order,
  then for ambiguous dates by article-insensitive alphabetical order
  (e.g. _A Graded Modal..._ comes after _Coeffects and..._).
* Use consistent formatting with the existing files.
